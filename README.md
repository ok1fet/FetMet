<h1>Fet MET Meteostanice</h1>
<p>je postavena na vývojové desce ESP32 LILYGO T3 v1.6.1, která má integrovaný LoRa modul (433 MHz) a</p>
<p>OLED displej. Díky tomu se cela konstrukce značně zjednodušuje.</p>
<ul>
<p>Pro měření směru a rychlosti větru, dešťových srážek, atmosférického tlaku, teploty a vlhkosti stačí </p>
<p>jen minimální počet součástek – konkrétně:</p>
<p>2x kondenzátory</p>
<p>1x rezistor</p>
<p>1x senzor BME280</p>
<ul>
<p>Data jsou přenášena prostřednictvím LoRa protokolu na server aprs.fi, kde je lze sledovat v reálném čase.</p>
<p>Senzory pro vítr a srážky jsou převzaty z komerčně dostupných meteostanic WH1080-90</p>
<p>, které lze snadno sehnat jako náhradní díly za přijatelnou cenu.</p>
