<h1>LoRa meteostanice</h1>
<p>je postavená na vývojové desce ESP32 LILYGO T3 v1.6.1, která má integrovaný LoRa modul (433 MHz) a</p>
<p>OLED displej. Díky tomu se celá konstrukce značně zjednodušila.</p>
<p>Pro měření směru a rychlosti větru, dešťových srážek, atmosférického tlaku, teploty a vlhkosti stačí </p>
<p>jen minimum součástek – konkrétně:</p>
<p>2x kondenzátory</p>
<p>1x rezistor</p>
<p>1x senzor BME280</p>
<p>Data jsou přenášena prostřednictvím LoRa APRS sítě na server aprs.fi, kde je lze sledovat v historická data.</p>
<p>Senzory pro vítr a srážky jsou použity z meteostanic WH1080-90, která jsou na trhu dostupná jako náhradní dil</p>
<p>za přijatelnou cenu.</p>
